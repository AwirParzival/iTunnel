- Ubuntu 18.04, Ubuntu 20.04, Ubuntu 22.04
- Tunnel & Cancel Tunnel
- AutoStart Tunnel
- Forward the desired port
- Forward as tcp
- Show IPv4 / IPv6
- SpeedTest
